My visionOS Apps

Pong Game

This is a classic pong game where you try to make the ball into a cup. When you start the game, you can choose the amount of cups that will be in the back row. This will dynamically add the rest of the cups and start the game. Using RealityKit and Reality Composer Pro, the game has realistic feeling physics when you throw the ball, hit a cup, or make it in the cup. Since it is a fully Immersive space, there is no limit to the space the game can take up.

Cube Ball Game

The game starts out with a ball on top of a cube in a Volume. Using a rotation gesture, you rotate the cube to roll the ball along it's edges. The goal is to find the hole and roll the ball into it. The ball appears to be magnetized to the cube because there is an invisible bigger cube that keeps the ball from falling off of the visible cube.
